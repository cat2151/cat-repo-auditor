# issue log表示モードをつける #75
[issues #75](https://github.com/cat2151/cat-repo-auditor/issues/75)

- これまでの課題： xキーを押しても反応がなくuserが混乱した。「これは未対応のrepo」「実行した」などのいずれかをlog領域表示で、その場でuserが確認したい、と感じた。

- 画面下半分にlog領域を表示する
- log領域のon/offは、SHIFT+Lでtoggleとする。デフォルトはoffであり、従来と同じ画面レイアウトとする。
- logはxキーを押すごとに、local config dirの log.txt に出力する。xキーで実行したcommandを表示する。1つの処理のlogを出力するごとにlogfileにflushする。logファイルと同じ内容を、画面下半分のlog領域に表示する。
